About Me

I’m a fourth-year Computer Science student at Simon Fraser University with two semesters left before graduation. I’m passionate about building scalable, efficient, and user-centered software. My experience spans web, mobile, and systems development — with hands-on projects in C++, Swift, and JavaScript. I enjoy solving complex problems through clean design, innovation, and continuous learning.

RafikiAI (iOS App)

RafikiAI is an iOS app built with Swift and SwiftUI that integrates the OpenAI Chat Completion API to deliver natural, conversational AI experiences. It assists users with everyday tasks such as creating fitness plans, generating meal ideas, and writing professional documents. This project strengthened my skills in SwiftUI design, API integration, and natural language processing.

IMDb Clone (Web App)

A fully responsive movie database application built using Next.js 14, Tailwind CSS, and the TMDb API. The app supports dark mode, dynamic routing, and server-side rendering for optimal performance. I learned modern React workflows, API-driven development, and deployment through Vercel.

ColorStackSFU (Community Platform)

ColorStackSFU is a community web platform built for Black and Latinx Computing Science students at SFU. Developed with HTML, CSS, JavaScript, and Firebase, it features user authentication, event management, and real-time updates. This project improved my skills in responsive web design, Firebase integration, and UI architecture.
